# 技术文档作品集

本仓库包含两个独立的技术文档项目，展示不同场景下的文档编写能力。

## 📁 项目一：平安医院挂号模块 API 文档（OpenAPI）
- **路径**：[registration-api/](./registration-api/)
- **内容**：门诊挂号系统 RESTful API 规范，涵盖科室、排班、预约、挂号记录等资源。
- **技术栈**：OpenAPI 3.0.3 / Swagger UI
- **使用**：浏览器打开 `index.html` 即可浏览和测试接口。

## 📁 项目二：Datasette 场景化入门指南
- **路径**：[stardew-guide/](./stardew-guide/)
- **主文档**：[datasette-stardew-guide.md](./stardew-guide/datasette-stardew-guide.md)
- **内容**：以星露谷物语作物数据为例，从零开始引导用户使用 Datasette 进行数据分析，涵盖安装、数据转换、SQL 查询等完整流程。
- **技术栈**：Datasette / SQLite / CSV / Markdown
- **配套文件**：示例数据 `stardew_crops.csv` 及操作截图 `images/`

## 📁 项目三：报销信息自动化处理系统 – 需求规格说明书
- **路径**：[requirement-analysis/](./requirement-analysis/)
- **内容**：培训讲师路费报销系统的需求分析文档，涵盖功能需求、字段定义、业务规则、验收标准及上线后迭代记录。
- **技术栈**：需求分析 / 产品设计 / 业务逻辑梳理
- **说明**：展示从痛点梳理到落地验收的完整需求规格，体现业务理解与文档规范能力。
